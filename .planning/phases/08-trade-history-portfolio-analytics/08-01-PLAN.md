<task id="08-01-T02" type="execute">
  <title>Add GET /api/portfolio/analytics endpoint</title>
  <read_first>
    - backend/app/routes/portfolio.py (existing _build_portfolio helper — use same price_cache pattern via request.app.state.price_cache)
    - backend/app/db.py (trades schema, positions schema: ticker, quantity, avg_cost; users_profile schema: cash_balance)
  </read_first>
  <action>
    In backend/app/routes/portfolio.py, add the following route after get_trades:

    @router.get("/analytics")
    def get_analytics(request: Request):
        price_cache = request.app.state.price_cache
        STARTING_CAPITAL = 10000.0

        with get_db() as conn:
            agg = conn.execute(
                """
                SELECT
                    COUNT(*) AS total_trades,
                    COALESCE(SUM(CASE WHEN side='buy'  THEN quantity*price ELSE 0 END), 0) AS total_invested,
                    COALESCE(SUM(CASE WHEN side='sell' THEN quantity*price ELSE 0 END), 0) AS total_received,
                    COALESCE(SUM(CASE WHEN side='buy'  THEN 1 ELSE 0 END), 0) AS buy_count,
                    COALESCE(SUM(CASE WHEN side='sell' THEN 1 ELSE 0 END), 0) AS sell_count
                FROM trades WHERE user_id=?
                """,
                (USER_ID,),
            ).fetchone()

            cash_row = conn.execute(
                "SELECT cash_balance FROM users_profile WHERE id=?", (USER_ID,)
            ).fetchone()

            positions = conn.execute(
                "SELECT ticker, quantity, avg_cost FROM positions WHERE user_id=?", (USER_ID,)
            ).fetchall()

            sell_trades = conn.execute(
                "SELECT t.ticker, t.price AS sell_price, p.avg_cost "
                "FROM trades t "
                "LEFT JOIN positions p ON p.user_id = t.user_id AND p.ticker = t.ticker "
                "WHERE t.user_id=? AND t.side='sell'",
                (USER_ID,),
            ).fetchall()

        total_invested = round(agg["total_invested"], 2)
        total_received = round(agg["total_received"], 2)
        cash_balance   = cash_row["cash_balance"] if cash_row else STARTING_CAPITAL

        # Exact realized P&L: accounting identity for single-user system
        realized_pnl = round(
            cash_balance - STARTING_CAPITAL - (total_invested - total_received), 2
        )

        unrealized_pnl = 0.0
        pnl_by_ticker = []
        for pos in positions:
            ticker   = pos["ticker"]
            qty      = pos["quantity"]
            avg_cost = pos["avg_cost"]
            price    = price_cache.get_price(ticker) or avg_cost
            unreal   = round(qty * (price - avg_cost), 2)
            pct      = round((price - avg_cost) / avg_cost * 100, 2) if avg_cost > 0 else 0.0
            unrealized_pnl += unreal
            pnl_by_ticker.append({"ticker": ticker, "pnl_percent": pct})
        unrealized_pnl = round(unrealized_pnl, 2)

        holdings_value = sum(
            pos["quantity"] * (price_cache.get_price(pos["ticker"]) or pos["avg_cost"])
            for pos in positions
        )
        total_value      = cash_balance + holdings_value
        total_return_pct = round((total_value - STARTING_CAPITAL) / STARTING_CAPITAL * 100, 2)

        best_performer  = max(pnl_by_ticker, key=lambda x: x["pnl_percent"]) if pnl_by_ticker else None
        worst_performer = min(pnl_by_ticker, key=lambda x: x["pnl_percent"]) if pnl_by_ticker else None

        sell_count = agg["sell_count"]
        winning_sells = sum(
            1 for st in sell_trades
            if st["avg_cost"] is not None and st["sell_price"] > st["avg_cost"]
        )
        win_rate = round(winning_sells / sell_count * 100, 1) if sell_count > 0 else 0.0

        return {
            "total_trades":     agg["total_trades"],
            "total_invested":   total_invested,
            "total_received":   total_received,
            "realized_pnl":     realized_pnl,
            "unrealized_pnl":   unrealized_pnl,
            "total_return_pct": total_return_pct,
            "best_performer":   best_performer,
            "worst_performer":  worst_performer,
            "win_rate":         win_rate,
            "buy_count":        agg["buy_count"],
            "sell_count":       sell_count,
        }
  </action>
  <acceptance_criteria>
    - GET /api/portfolio/analytics returns HTTP 200
    - Response contains all 11 keys: total_trades, total_invested, total_received, realized_pnl, unrealized_pnl, total_return_pct, best_performer, worst_performer, win_rate, buy_count, sell_count
    - best_performer and worst_performer are either null (no open positions) or objects with keys: ticker (string) and pnl_percent (number)
    - On fresh portfolio (zero trades): all numeric fields = 0 or 0.0, best_performer=null, worst_performer=null
    - win_rate is 0.0 when sell_count=0 (no division by zero)
    - Endpoint uses request.app.state.price_cache (same pattern as existing get_portfolio route)
  </acceptance_criteria>
</task>
